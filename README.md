# Project Polaris

Project Polaris is an end-to-end enterprise analytics platform built on Microsoft Fabric for NimbusFlow, a fictional SaaS workflow automation company.

The project demonstrates how data from CRM, subscription billing, product usage, marketing and customer support systems can be ingested into OneLake, transformed through a medallion architecture and served through curated analytical models for Power BI reporting.

All datasets used in this project are synthetic and were created exclusively for learning and portfolio purposes. The project contains no employer, customer or production data.

## Business Problem

NimbusFlow's operational data is distributed across multiple systems. Sales, Finance, Product and Customer Support teams currently calculate key metrics independently, resulting in inconsistent definitions of customers, recurring revenue, churn and product adoption.

The objective is to build a governed Microsoft Fabric analytics platform that provides trusted, reusable data for executive and departmental reporting.
